## 📕 About

Devfinder is a Github developer profile searcher.

---

## ⚒️ Used Technologies

To develop this app, I used these technologies:

- React
- Typescript

---

## ✨ Features

- Responsivity
- Developer profile searching using the Github API.
- PWA features, such as offline support.

---

## ⚙️ How to Use

First, clone the repository.

Inside the project folder, run `yarn` or `yarn install` to install the dependencies.

Then, run `yarn start` to start a local development server.
